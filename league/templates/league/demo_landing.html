About This Project

A full-stack fantasy baseball platform built for Division III college baseball.

  • GitHub Actions scheduler
Features

  • Pickups, trades, weekly lineups
  • Head-to-head matchups + standings
  • Stat dispute system
Demo Mode

  • Team names are anonymized
  • No changes permitted, read-only access
