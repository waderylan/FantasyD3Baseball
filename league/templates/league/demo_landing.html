About This Project

A full-stack fantasy baseball league built for Division III college baseball.

  • Tech Django and Python, deployed on PythonAnywhere
  • Live stats box scores scraped automatically from official Liberty League results using Playwright and BeautifulSoup, running on a schedule via GitHub Actions
  • Full league management including free agent pickups, trades, weekly lineup setting, head-to-head matchups, and standings
  • Dispute system lets teams flag incorrect stats for commissioner review and resolution
  • Live data the demo shows the actual league database with stats, rosters, and standings reflecting the current state of the season
  • Concurrent users with multiple teams able to be logged in and active simultaneously
  • Read-only demo with team names anonymized and no changes permitted
